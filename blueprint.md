# Lotto Studio Blueprint

## 📋 프로젝트 개요: "Lotto Studio Open Lab"

- **슬로건:** "AI가 설계한 27,132개의 한정판 행운."
- **목표:** 5개 앙상블 AI 모델이 생성한 정제된 번호 조합을 제공하고, 게임화된 사용자 경험(UX)을 통해 로또 번호 선택 과정을 즐겁게 만듭니다.
- **현재 상태:** 핵심 기능(Hero, Generator, Insight, League)이 구현된 동적 1페이지 웹 애플리케이션. Annuity 섹션 기능 구현을 앞두고 있습니다.

---

## ✨ 구현된 기능 상세

### 1. **1페이지 스크롤 레이아웃 (`index.html`)**
- 사용자의 스크롤 흐름에 따라 Lotto Studio의 모든 핵심 기능을 탐색할 수 있는 단일 페이지 애플리케이션 구조를 완성했습니다.
- **구성 요소:** Hero, Generator, Insight, League, Annuity 섹션의 기본 HTML 구조가 모두 포함되어 있습니다.

### 2. **모던 & 대담한 디자인 (`style.css`)**
- "Bold Definition" 디자인 가이드라인에 따라 생동감 넘치는 색상 팔레트, 깊이감을 주는 그림자 효과, 뛰어난 가독성의 타이포그래피를 적용했습니다.
- **핵심 스타일:** `var()`를 사용한 CSS 변수 시스템, 재사용 가능한 컴포넌트 스타일, 모바일 반응형 디자인을 완벽하게 구현했습니다.

### 3. **Hero 섹션: AI 위원회 소개**
- 5명의 AI 위원회(루나, 라이트, 로기, SVM, XGB) 캐릭터 이미지를 세련된 그리드 레이아웃과 함께 표시합니다.
- 각 캐릭터 카드는 부드러운 호버 효과를 통해 사용자에게 인터랙티브한 경험을 제공합니다.

### 4. **Generator 섹션: 3가지 게임 모드 선택**
- "럭키 박스", "금고 해킹", "타로 카드" 3가지 게임 모드를 선택할 수 있는 인터랙티브 UI를 구현했습니다.
- 각 모드 카드는 독특한 아이콘, 제목, 설명을 포함하며, 클릭 시 해당 모드가 선택되었음을 알리는 이벤트 리스너가 `main.js`에 추가되었습니다.

### 5. **Insight 섹션: 후보 번호 히트맵**
- `combined_priority_numbers.txt`에서 19개의 후보 번호 데이터를 동적으로 불러옵니다.
- Chart.js 라이브러리를 사용하여 후보 번호의 득표수를 시각적으로 표현하는 아름다운 가로 막대 차트(히트맵)를 렌더링합니다.

### 6. **League 섹션: 모델별 성적 및 명예의 전당**
- `league_data.json` 파일에서 지난주 성적 데이터를 비동기적으로 불러와 동적으로 UI를 구성합니다.
- **명예의 전당:** 가장 뛰어난 성과를 낸 모델을 특별히 강조하여 보여줍니다.
- **성적 표:** 각 모델의 순위, 예측 번호, 적중 개수를 테이블 형태로 제공하며, 지난주 당첨 번호와 일치하는 예측 번호는 하이라이트 처리하여 가독성을 높였습니다.

## 🚀 현재 및 다음 단계: Annuity 섹션 구현

**목표:** 메인 페이지 하단에 "로기(거북이)의 연금복권 최적화 조합" 섹션을 구현하여, 사용자에게 또 다른 형태의 행운 조합을 제공합니다.

| **작업 항목** | **세부 실행 계획** |
| :--- | :--- |
| **1. UI 구조 설계 (`index.html`)** | - Annuity 섹션 내에 조합 결과를 표시할 컨테이너와 번호 생성 버튼을 추가합니다.<br>- "로기(거북이)" 캐릭터 이미지를 배치하여 섹션의 테마를 강조합니다. |
| **2. 동적 스타일링 (`style.css`)** | - Annuity 섹션에 맞는 독특하고 매력적인 디자인을 적용합니다.<br>- 생성된 번호가 눈에 잘 띄도록 시각적 효과를 추가하고, 전체적인 디자인 통일성을 유지합니다. |
| **3. 조합 생성 로직 (`main.js`)** | - "로기의 전략"(끝자리 중복 제거 등)에 기반한 연금복권 조합 생성 함수를 작성합니다.<br>- 생성 버튼 클릭 시, 이 함수를 호출하여 7자리 연금복권 번호를 생성하고, UI에 동적으로 렌더링합니다. |
